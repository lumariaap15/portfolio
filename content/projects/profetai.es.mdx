Un sistema de seguimiento de precios para el mercado argentino. Consume datos abiertos de SEPA, normaliza nombres de productos heterogéneos con embeddings y presenta resúmenes de precios generados por LLM — con una capa de notificaciones por Telegram para alertas de bajadas de precio.

El principio de diseño que lo guía: **la lógica determinística se mantiene determinística** (parsing, umbrales de matching, alertas) y el LLM se reserva para las tareas genuinamente difusas (normalización, resumen).
